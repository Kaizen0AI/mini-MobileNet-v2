# CIFAR-10 Mini_MobileNetV2

This project is a CNN-based MobileNetV2 built using TensorFlow.

## Problem
Classify images as:
- All 10 classes in CIFAR-10 dataset

## Model
- Conv2D
- Batch Normalization
- ReLU
- MaxPooling
- Dense
- Depthwise Con2D
- Globle Average pooling
- Data Augmentation

## Results
### Run 1
- Baseline Mini-MobileNetV2
- Test Accuracy: 65.5%

### Run 2
- Added data augmentation
- Test Accuracy: 66.7%
