# CIFAR-10 Mini_MobileNetV2

This project is a CNN-based MobileNetV2 built using TensorFlow.

## Problem
Classify images as:
- All 10 classes in CIFAR-10 dataset

## Model
- Conv2D
- Batch Normalization
- ReLU
- MaxPooling
- Dense
- Depthwise Con2D
- Globle Average pooling
- Data Augmentation

## Experiments

### Baseline
- Accuracy: 65.5%

### Experiment 1
- Added data augmentation
- Accuracy: 66.7%

### Notes
- Reduced overfitting
- Improved validation accuracy
